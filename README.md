# Flight Price Analysis Dashboard (Power BI)

This project presents an interactive Power BI dashboard designed to analyze flight pricing trends, route performance, layover types, and airline distribution.

## Key Insights

• Average Flight Price Analysis  
• Route-wise Price Comparison  
• Layover Type Price Distribution  
• Airline-wise Filtering  
• Monthly Price Trend Analysis  

## Dashboard Features

- Interactive Airline Filters
- Route Price Analysis
- Layover Type Price Breakdown
- Monthly Flight Price Trend
- KPI Metrics for Price Statistics

## Tools Used

Power BI  
Data Modeling  
DAX Measures  
Interactive Filters  

## Dataset

Flight price dataset including airline, route, layover, and pricing information.

## Use Case

This dashboard helps analyze:

- Price patterns across airlines
- Popular routes and their pricing
- Layover impact on flight prices
- Monthly pricing trends
